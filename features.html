Scene features

Feature Image

The scene terrain is being generated within the shaders, using a height map to add different altitude levels to the terrain plane.

To create the height map, the normal map and the texture terrain, the tool L3DT has been employed .

To create the forest, a Perlin noise algorithm has been used. For that, a representation area has been defined, the cliff area; and also an exclusion area, the path which goes from the limit of the forest to the lighthouse.

In addition, the terrain height map image has been loaded, in order to define the 'Y' coordinate for each tree.

~1min Day / Night cycle. The directional light which serves as the sun, rotates around the scene center, achieving a light Day / Night effect, which affects different scene objects: the terrain, the trees, the lighthouse, the sea...

The sea has ripples generated by a noise function inside the shader, reflects the sun light, the seabed is visible through it and also changes its color depending on the light incidence.

Special thanks to Jonas Wagner

The lighthouse structure has a shiny old metal effect, which reflects the sunlight. It is also affected by two different lights, the directional sunlight and the torch point light.

The point light and the torch, rotate with the same angle.

The user can interact with the lighthouse torch, via the different controls located at the bottom of the demo page.

This last functionality is employing Tanner Helland's algorithm to make the conversion between Kelvin units and RBG values.

A global fog atmosphere effect has been added to each scene object, to increase the realism of the scene.
